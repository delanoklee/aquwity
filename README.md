# Acuity

A simple desktop app with a text input bar.

## Download

Download for your platform from the [Releases page](https://github.com/delanoklee/aquwity/releases).

- **Windows**: Download `Acuity.exe` and double-click to run
- **macOS**: Download the DMG, open it, drag Acuity to Applications
- **Linux**: Download the AppImage, make executable (`chmod +x`), and run

## Development

```bash
# Install dependencies
pip install -r requirements.txt

# Run
python main.py

# Build
pyinstaller --clean acuity.spec
```
