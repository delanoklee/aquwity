FAQ

Is Acuity really free?

Yes! The app is free. You only pay for the AI analysis through Anthropic's API (~$1.50/day for an 8-hour workday).

How does it protect my privacy?

Screenshots are sent to Claude AI for analysis, then immediately deleted. Only the analysis results are stored locally.

What's an API key?

It's like a password that lets Acuity use Claude AI to analyze your screenshots. You get one from Anthropic in 2 minutes.

Can I adjust the screenshot interval?

Yes! The default is 5 minutes, but you can change it in the settings to reduce costs or increase monitoring frequency.
